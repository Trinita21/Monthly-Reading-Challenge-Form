Monthly Reading Challenge

We have 4 different themes (will have different channels for each and the idea is to read one book a month):

  1. adulting - anything that you know will help you somehow, like atomic habits, the defining decade, becoming, lean in, educated, etc.
  2. escape - classic fiction/scifi go here
  3. fun - children's literature, graphic novels, humor books
  4. curiosity - this is for avid readers, picking up an out of comfort zone book, different opinion, long/academic books like architecture books, game design, etc.

I'd recommend to pick only 1 for now (thinking to put some monetary friction to join multiple clubs at the same time).

You'll get the link to join the community Discord server after you fill out this form. See you there!
